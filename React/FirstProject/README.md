# Event Planner for Teams

A simple React + TypeScript app to create events, add members, and manage RSVPs (Yes / No / Maybe).  
All data is stored in LocalStorage, so it works without any backend.

This project is built to practice core React concepts like hooks, routing, context, reducer, and basic DOM manipulation.

## Features
- Create and list events  
- View event details  
- Add members  
- RSVP system (Yes / No / Maybe)  
- Live stats  
- Data persistence using LocalStorage
